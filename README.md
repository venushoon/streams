
### ✨ 디자인 및 사용자 경험 (UX/UI)
* **뉴모피즘(Neumorphism) 스타일**: 부드러운 그림자와 입체감을 활용하여 눈의 피로를 줄이고, 학생들이 마치 실제 버튼을 누르는 듯한 물리적인 피드백을 느끼게 설계했습니다.
* **전자칠판 최적화**: 대형 화면에서 하단 메뉴가 잘리지 않도록 레이아웃을 계산했으며, 터치 간섭을 최소화하기 위해 버튼 간격과 크기를 넉넉하게 배치했습니다.
* **애니메이션 효과**: `Animate.css`를 활용하여 숫자 등장, 타일 배치, 결과 발표 시 역동적인 움직임을 주어 학생들의 몰입도를 높였습니다.

### 🎮 핵심 게임 기능
* **다인용 대전 모드 (1~4인)**: 한 화면을 분할하여 최대 4명의 학생이 동시에 자기 보드에 숫자를 배치하며 경쟁할 수 있습니다.
* **지능형 숫자 주머니**: 원작 규칙을 반영하여 1~10, 20~30은 각 1개씩, 11~19는 각 2개씩, 그리고 강력한 변수인 **조커(★)** 타일 1개를 포함한 총 40개의 타일 덱을 무작위로 생성합니다.
* **실시간 배치 동기화**: 모든 플레이어가 숫자를 배치해야 다음 숫자가 나오는 방식을 채택하여, 실력 차이가 나더라도 게임 속도가 한 명에게 치우치지 않게 조절했습니다.

### 📚 교육적 장치 및 편의성
* **인터랙티브 가이드**: 시작 화면에 '게임 방법' 팝업을 배치하여 선생님이 별도로 규칙을 설명하지 않아도 학생들이 스스로 이해하고 시작할 수 있습니다.
* **자동 점수 계산**: 스트림스 특유의 복잡한 마디 점수 계산(1개 0점, 20개 연결 시 300점 등)을 알고리즘이 즉시 처리하여 게임 종료 후 바로 순위를 보여줍니다.
* **상태 초기화 시스템**: 여러 판을 반복해서 즐길 수 있도록 '다시 하기'와 '홈으로' 기능을 통해 모든 게임 데이터를 깨끗하게 리셋하는 로직을 갖추고 있습니다.
